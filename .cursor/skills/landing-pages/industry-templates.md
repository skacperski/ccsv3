# Industry section templates

Section outlines for common verticals. Adapt copy to `.agents/product-marketing-context.md` when available. Remove sections that don’t serve the single conversion goal.

## SaaS (trial / demo)

**Goal:** Start trial or book demo  
**Type:** Lead gen or clickthrough  
**Length:** Medium–long for B2B

| Section | Content |
|---------|---------|
| Hero | Outcome headline; subhead for role; CTA “Start free trial” / “Book demo”; product screenshot or people+UI |
| Social proof strip | Logos or single stat (LinkedIn pattern) |
| Problem | 2–3 bullets on workflow pain |
| Solution | How product delivers outcome (3 steps) |
| Features → benefits | 3–4 cards; lead with benefit |
| Proof | Testimonial + case study snippet |
| FAQ | Collapsible; security, pricing, cancel |
| Final CTA | Repeat hero offer |

**Patterns:** SEMrush (narrow focus), LinkedIn (jump links), Gusto/Mailchimp (clear modules)

## SaaS (single feature / campaign)

**Goal:** Signup for one capability (e.g. competitor analysis)  
**Length:** Short–medium

| Section | Content |
|---------|---------|

| Proof | Logos, awards, tweets |
| 2–3 benefits | Audience-specific (marketers, data) |
| CTA repeat | Same action |
