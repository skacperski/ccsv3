# Landing page reference checklist

Extended QA and FAQ patterns from landing page fundamentals (Unbounce).

## Conversion experience checklist

### Relevance
- [ ] Content laser-focused on one offer
- [ ] Consistent with path before landing (ad, email, social post)
- [ ] No generic homepage copy pasted in

### Performance
- [ ] Target ~1–2 second load where possible
- [ ] Optimized images (WebP/AVIF, dimensions, lazy below fold)
- [ ] Minimal third-party scripts on conversion pages

### Navigation
- [ ] No main site nav (or minimal: logo only if required)
- [ ] No footer with 10+ links
- [ ] Internal links only if they support the single CTA

### Mobile
- [ ] Responsive, not just “shrunk desktop”
- [ ] CTA full-width or thumb-reachable
- [ ] Form inputs large enough; correct `inputmode` / types

## How many landing pages?

| Factor | Guidance |
|--------|----------|
| Campaigns | At least one page per major campaign |
| Segments | Separate pages when pain/offer differs materially |
| Products | One per product or category when benefits differ |
| Testing | Variants for A/B or multivariate tests |

More targeted pages generally outperform one generic destination.

## FAQ — quick answers

**Can any page be a landing page?**  
In analytics, any destination URL counts. For marketing, insist on: standalone, **one CTA**, **no website navigation**.

**Landing page without a full website?**  
Yes — common for launches, tests, and time-boxed offers.

**How long should the page be?**  
As long as needed for the visitor to take action — no more. Simple offer → short. Complex B2B → longer with proof.

**Video on landing pages?**  
Use when it clarifies the offer (demo, product in use). Autoplay with sound off; don’t delay CTA visibility.

**Homepage vs landing page**  
Homepage: explore brand, many paths. Landing page: one path to convert.

## Anti-patterns

- Sending all PPC traffic to homepage
- Multiple primary CTAs (“Buy” + “Book demo” + “Newsletter”)
- Long forms for top-of-funnel cold traffic
- Company history and team bios on campaign pages
- Slow hero sliders and heavy video autoplay above fold
- Broken message match (ad says “50% off”, page shows list price)
